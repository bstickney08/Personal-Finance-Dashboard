# Personal-Finance-Dashboard
DAT-230 Final Project

- External library: matplotlib or plotly

- Dataset: CSV with expenses

- Visualization: Spending categories

- Common good angle: Encourages financial literacy

Description:
A simple tool that allows users to upload a CSV file of their monthly spending. The dashboard automatically categorizes expenses (such as food, bills, and entertainment) and displays clear visual charts to help users understand their spending habits and make better financial decisions.


# How to Organize Files on GitHub
